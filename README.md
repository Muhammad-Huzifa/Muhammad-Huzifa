Designed and implemented a **Temporal Transformer Spatio-Temporal Graph Convolutional Network (TT-STGCN)** ‚Äî a lightweight, high-performance architecture for gesture-based sign language recognition.

**Key Innovations:**
- ‚úÖ **Pose-based Modeling**: Leverages MediaPipe landmarks for efficient feature extraction
- ‚úÖ **Hybrid Architecture**: Combines Graph Convolutional Networks with Temporal Transformers
- ‚úÖ **Scalable Design**: Trained on 300 classes with inference optimization for 100-class subsets
